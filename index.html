Funcionalidades

Cada categoria de comandos foi pensada para tornar o teu servidor mais ativo e fácil de gerir.

🎵

Música

Reproduz as tuas músicas favoritas diretamente nos canais de voz do servidor.

!play!skip!queue
🎮

Diversão

Comandos divertidos para interagir com os membros e animar as conversas.

!kiss!say
🛡️

Administração

Ferramentas simples de moderação para manter o servidor organizado.

!clear!prefix
ℹ️

Informações

Consulta dados sobre utilizadores, o servidor e o próprio bot.

!userinfo!serverinfo!botinfo
✳ Referência rápida

Lista de comandos

Usa o prefixo ! antes de cada comando.

Música

  • ✳ !play [nome/link]
  • ✳ !skip
  • ✳ !queue

Diversão

  • ✳ !kiss @utilizador
  • ✳ !say [mensagem]

Administração

  • ✳ !clear [quantidade]
  • ✳ !prefix [novo prefixo]

Informações

  • ✳ !userinfo @utilizador
  • ✳ !serverinfo
  • ✳ !botinfo
